# AI Storyboard Director v5.2

AI Storyboard Director v5.2 是一套面向 AIGC 视频制作的导演级分镜技能。它不是简单按剧本文字切段，而是先判断故事终点、人物关系、情绪路径和视觉概念，再输出可审查、可修改、可交给下游生成工具执行的镜头链。

## 文件说明

| 文件 | 用途 | 适合场景 |
|---|---|---|
| `ai-storyboard-director5.2.skill` | 可安装技能包，内含 `SKILL.md`、知识库和示范资产 | 用于 Codex/支持 `.skill` 导入的 Agent 环境 |
| `ai-storyboard-director5.2-全量单文件.md` | 完整自包含版，内联主指令、15 个知识参考文件和示范集 | 任意大模型、长上下文模型、需要完整知识深度的分镜工作 |
| `ai-storyboard-director5.2-精编单文件.md` | 小上下文精编版，保留输出契约、铁律、受控词表和核心决策程序 | 小上下文模型、快速拆分镜、移动端或轻量工作流 |

## 核心能力

- 剧本到分镜：读取剧本后，按导演决策骨架拆成连续 AG-CLIP 镜头链。
- 设计稿 + 生产稿双速交付：先给人看的导演方案，再展开为给机器执行的全字段生产稿。
- 视觉概念前置：每章先锁定视觉概念、母题、空间权力几何、反常规决定和风格坐标。
- 剪辑动机留痕：每一剪用受控 token 标出新信息、动作、反应、声音、视线、节奏或情绪动机。
- 审查模式：可检查已有分镜的问题，包括连续性、物理可生成性、搬运原文、导播病等。
- AIGC 可执行约束：输出只保留物理事实，避免比喻、散文解释和下游不可生成描述。

## 快速使用

### 技能包

将 `ai-storyboard-director5.2.skill` 导入支持 `.skill` 的 Agent 环境后，直接发送剧本并说明“拆分镜”。技能会先生成设计稿，确认后再展开生产稿；如果需要跳过设计稿，可明确说“直接出生产稿/契约格式”。

### 单文件版

把对应 Markdown 文件整体作为系统提示词或对话首条消息发送给模型，然后发送剧本并要求“拆分镜”。

- 需要完整知识库：使用 `ai-storyboard-director5.2-全量单文件.md`
- 需要轻量快速执行：使用 `ai-storyboard-director5.2-精编单文件.md`

## v5.2 重点更新

- 新增“立概念”步骤：先回答“这一章作为画面是什么主意”，再切节拍。
- 新增“双速交付”默认流程：设计稿给人拍板，生产稿给机器执行。
- 新增创造性自检：导播测试、概念三镜引证、反常规执行对账、母题链完整性。
- 输出契约保持兼容：与 v4.5、v5.0、v5.1 的生产稿结构兼容。

## 适用对象

- AI 短片、短剧、广告片、概念片分镜设计
- 即梦、可灵、Runway、Pika、Seedance 等视频生成工作流前置拆镜
- 影视创作者、AIGC 导演、提示词工程师、分镜审核人员
- 需要把剧本转为可执行镜头清单的 Agent 管线
